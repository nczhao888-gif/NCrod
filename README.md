# NC Rod Basic Static Website

A lightweight, GitHub-ready static website starter based on the public structure and business positioning of [NC Rod](https://www.ncrod.com/). It is suitable for GitHub Pages, simple company profile hosting, or as a front-end base for a future WordPress/headless CMS rebuild.

## Project Purpose

This repository provides a clean foundation for an industrial B2B website focused on nickel-based alloy rods and round bars, including:

- Company introduction
- Product category cards
- Alloy application highlights
- Quality and manufacturing capability sections
- Contact call-to-action
- Basic SEO metadata
- Responsive layout
- Simple JavaScript interactions

## Website Positioning

NC Rod is positioned as a supplier of nickel-based alloy rods and round bars, covering material families such as Inconel, Hastelloy, Incoloy, Monel, Pure Nickel, Nimonic, and Invar. The content also highlights customized dimensions, processing capability, quality control, and international standards such as ASTM, ASME, GB, DIN, and EN.[NC Rod](https://www.ncrod.com/)
